# Sublime3 C/C++ 插件
1. SublimeAStyleFormatter
整理美化编写的C/C++代码，快捷键Ctrl+Alt+F

2. ConvertToUTF8
此插件可以有效的解决中文乱码问题

3. BracketHighlighter
高亮选中代码对应的括号

4. SublimeLinter
检查代码语法是否有错误，并提示

5. Alignment
对齐代码，快捷键Ctrl+Alt+A

6. CTags
产生标记文件以帮助在源文件中定位对象

7. All Autocomplete
让代码自动完成的匹配从所有打开的文件里去匹配，而不是只在当前文件里匹配

8. Git
Git管理

9. SideBarGit 
侧边栏 Git 支持

10. AdvancedNewFile 
通过输入路径来创建文件，快捷键是 Ctrl + Alt + N
